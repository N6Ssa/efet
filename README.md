# EFET Studio — точная копия сайта

Локальная копия [efet-studio.ru](https://efet-studio.ru/) — оригинальный HTML/CSS/JS из Tilda.

## Запуск с live reload

Дважды кликните **`start.bat`** или в терминале:

```powershell
powershell -ExecutionPolicy Bypass -File dev_server.ps1
```

Откройте **http://127.0.0.1:5173** — при сохранении `index.html` страница перезагрузится автоматически.

## Структура

| Файл | Описание |
|------|----------|
| `index.html` | Полная страница сайта (Tilda, ~1.2 МБ) |
| `original.html` | Резервная копия с сайта |
| `dev_server.ps1` | Локальный сервер с автоперезагрузкой |
| `start.bat` | Быстрый запуск сервера |

## Как редактировать

Сайт свёрстан в **Tilda Zero Block** — каждый блок имеет id вида `rec1921353361`.

- **Тексты** — ищите нужную фразу в `index.html` (Ctrl+F) и меняйте прямо в файле.
- **Стили блока** — inline `<style>#rec...` рядом с каждым блоком.
- **Картинки** — URL на `static.tildacdn.com`; замените на свои или положите в `public/` и обновите путь.
- **Формы** — отправляются через Tilda Forms (нужен интернет).

## Шрифты

На сайте используются **NEUTRALFACE** и **Geometria** — подгружаются с CDN Tilda автоматически.

## Примечание

Для максимального совпадения с оригиналом используется исходный HTML Tilda, а не упрощённая вёрстка. Это даёт идентичный вид, но файл большой — для крупных правок удобнее искать текст через поиск в редакторе.
